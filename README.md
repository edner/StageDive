# Stage Dive 🤘

A high-intensity concert simulation game built with Phaser 4, TypeScript, and Vite. 

In **Stage Dive**, you play as an energetic fan trying to pull off the ultimate stunt: a massive stage dive. Build up your HYPE meter, evade relentless roadies, and interact with the band members on stage to achieve the highest score possible before launching yourself into the crowd!

## Features

- **Dynamic Concert Action:** Navigate the stage while avoiding roadies who are trying to throw you out.
- **HYPE System:** Interact with the band members (singer, guitarist, drummer, bassist) to build your HYPE meter.
- **Interactive Band Members:** Jamming with musicians gives you bonus points and temporarily resets the roadies' pursuit!
- **The Ultimate Stage Dive:** Once your HYPE is maxed out, hit the **Spacebar** to leap into the crowd and finish the level in style.
- **Neon Aesthetic:** High-energy visual design with glowing elements and a dark, moody concert atmosphere.

## Tech Stack

- **[Phaser 4](https://phaser.io/)** - The fast, free, and fun open-source HTML5 game framework.
- **[TypeScript](https://www.typescriptlang.org/)** - For type-safe and robust game logic.
- **[Vite](https://vitejs.dev/)** - Next-generation frontend tooling for a lightning-fast development experience.

## Getting Started

### Prerequisites

Make sure you have [Node.js](https://nodejs.org/) installed on your machine.

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/stagedive.git
   cd stagedive
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Open your browser and navigate to the local URL provided by Vite (usually `http://localhost:5173`).

### Building for Production

To create a production build of the game, run:

```bash
npm run build
```

This will generate a `dist` folder containing the compiled, minified code ready to be deployed to any static hosting service (GitHub Pages, Vercel, Netlify, etc.).

## How to Play

- **Movement:** Use the **Arrow Keys** or **WASD** to move around the stage.
- **Interact:** Get close to band members to jam with them and build HYPE faster.
- **Evade:** Keep your distance from the roadies. If they catch you, it's game over!
- **Stage Dive:** When the HYPE meter is completely full, press **Spacebar** to perform the stage dive and win!
